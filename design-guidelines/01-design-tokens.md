# 01. 디자인 토큰 — 색상 · 타이포그래피 · 간격 · 그림자

프로모터스 UI의 모든 화면이 공유하는 기본 값입니다. 새 화면·컴포넌트를 만들 때
여기 정의된 값만 사용하면 기존 화면과 자동으로 일관성이 맞습니다.

## 1. 색상 팔레트

`style.css`의 `:root`에 선언된 CSS 변수:

```css
:root {
  --navy:      #1e2464;  /* 브랜드 메인. 제목, 강조 텍스트, 진한 버튼 배경 */
  --yellow:    #f5c400;  /* 브랜드 포인트. 활성 밑줄, 네이비 버튼 위 텍스트 */
  --gold:      #c9a544;  /* 금색 띠(사선 테두리), 프리미엄 연출 */
  --gray:      #9aa5b1;  /* 보조 회색 */
  --red:       #c0392b;  /* 경고, 취소, 미결제 */
  --blue:      #2f6fe4;  /* 마이페이지/앱 영역 메인 (토스 스타일) */
  --blue-dark: #1b4fc0;  /* 파란 그라데이션의 어두운 쪽 */
  --blue-soft: #eaf2ff;  /* 파란 아이콘 배경, 연한 파랑 */
}
```

변수 외에 관례적으로 쓰는 고정 색:

| 용도 | 값 |
|------|-----|
| 페이지 배경 | `#e9ebee` (쿨그레이) |
| 본문 텍스트 | `#2b2f33` (순검정 금지) |
| 보조 텍스트 | `#555f6b` |
| 흐린 텍스트 (메타, 힌트) | `#8a94a0` |
| 카드 테두리 | `#e2e6ea` 1px |
| 입력창 테두리 | `#d6dbe1` 1px |
| 구분선 | `#eef1f4` |
| 금색 텍스트 (금액·강조) | `#b8963a` |
| 성공/네이버 그린 | `#03c75a`, 텍스트는 `#038550`, 배경 `#e6f7ee` |
| 경고 배경 | `#fdecea` |
| 포스트잇(메모) | 배경 `#fff9c8`~`#fffdf0`, 테두리 `#ead45c` |
| 알림 뱃지 | `#ef4444` |

**색 영역 구분 원칙**: 공개 홈페이지 영역(소개·오시는길·정비사례·공지)은
**네이비+옐로+골드**, 마이페이지·모바일 앱 영역은 **블루 계열**을 사용합니다.
두 영역의 색을 섞지 마세요.

## 2. 타이포그래피

```css
font-family: 'Noto Sans KR', 'Malgun Gothic', sans-serif;
```

- Google Fonts에서 `wght@400;500;700;900` 4단계를 로드합니다.
- 굵기 사용 규칙:
  - **900 (Black)**: 초대형 타이틀("정비예약" 125px), 섹션 키커, 숫자 뱃지
  - **700 (Bold)**: 버튼, 탭, 내비게이션, 카드 제목
  - **800~900**: 작은 UI 텍스트를 강하게 (링크, 라벨, em/strong)
  - **500 (Medium)**: 설명 문단
  - **400**: 긴 본문
- 대형 제목은 자간을 조입니다: `letter-spacing: -1px ~ -2px`
- 대형 제목에는 은은한 네이비 그림자: `text-shadow: 3px 4px 6px rgba(30,36,100,.18)`
- 본문 행간 1.55~1.75, 카드 제목 행간 1.35
- 한글 줄바꿈: 제목류는 `word-break: keep-all`

대표 크기 (데스크톱 기준):

| 요소 | 크기 |
|------|------|
| 히어로 타이틀 | 125px / 900 (반응형: `min(clamp(64px, 6vw, 118px), 11vh)`) |
| 화면 제목 (intro h2) | 72px / 900 |
| 섹션 키커 | 28px / 900 |
| 설명 문단 | 28px / 500 |
| 카드 제목 | 18~19px / 700 |
| 본문 | 14~16px |
| 메타/힌트 | 12~13px |

## 3. 모서리 라운딩

| 요소 | 반경 |
|------|------|
| 알약(필터 칩, 뱃지, 스텝) | `999px` |
| 마이페이지 카드 (앱 느낌) | `16~22px` |
| 모달 | `14px` |
| 일반 카드 (post, notice, cust) | `12px` |
| 지점 카드, 드롭다운, 캘린더 요소 | `8~10px` |
| 버튼, 입력창 | `8px` |
| 미니 버튼, 썸네일 | `6px` |

## 4. 그림자

그림자 색은 순검정 대신 **브랜드색 계열의 반투명**을 사용합니다.

```css
/* 일반 카드 (홈페이지 영역: 네이비 톤) */
box-shadow: 0 6px 18px rgba(30,36,100,.07);

/* 카드 호버 (살짝 떠오름과 함께) */
box-shadow: 0 10px 26px rgba(30,36,100,.12);
transform: translateY(-2px);

/* 드롭다운/플로팅 */
box-shadow: 0 12px 28px rgba(30,36,100,.16);

/* 모달 */
box-shadow: 0 24px 64px rgba(0,0,0,.3);

/* 마이페이지 카드 (앱 영역: 슬레이트 톤) */
box-shadow: 0 6px 18px rgba(15,23,42,.06);

/* 파란 카드 (그 색 자체의 그림자) */
box-shadow: 0 14px 30px rgba(27,79,192,.28);

/* 입체(눌리는) 버튼: 아래에 단색 두께 + 확산 그림자 */
box-shadow: 0 6px 0 var(--yellow), 0 12px 24px rgba(30,36,100,.3);
/* hover 시 translateY(2px) + 그림자 두께를 4px로 줄여 눌리는 느낌 */
```

## 5. 간격

- 카드 그리드 간격: `18px` (모바일 `12~14px`)
- 카드 내부 패딩: `16~20px`
- 섹션 사이 큰 여백은 `1cm`, `1.5cm` 단위를 사용 (이 프로젝트의 특징적 관례)
- 데스크톱 히어로 좌측 패널은 화면 높이에 비례:
  `clamp(24px, 4vh, 48px)` 식으로 vh 기반 clamp 사용

## 6. 그라데이션 레시피

```css
/* 파란 카드 (마이페이지 차량/헤더 카드) */
background:
  radial-gradient(circle at 80% 0%, rgba(255,255,255,.18), transparent 42%),
  linear-gradient(135deg, var(--blue) 0%, var(--blue-dark) 100%);

/* 네이비 썸네일 placeholder */
background: linear-gradient(135deg, #273261 0%, #1e2464 55%, #161a4a 100%);

/* 금색 띠 */
background: linear-gradient(180deg, #ecd98f 0%, var(--gold) 45%, #a8862e 100%);

/* 실버(새틴) 시트: 베이스 그라데이션 + 사선 광택 띠 겹치기 */
background:
  linear-gradient(115deg,
    transparent 20%, rgba(255,255,255,.85) 26%, transparent 32%,
    transparent 44%, rgba(200,205,214,.35) 50%, transparent 56%,
    transparent 64%, rgba(255,255,255,.9) 72%, transparent 80%),
  linear-gradient(115deg,
    #ffffff 0%, #f3f4f6 22%, #ffffff 40%,
    #e9ebef 58%, #f8f9fa 75%, #ffffff 100%);
```

## 7. 전환(transition)

- 기본: `.15s ~ .2s`, 속성 명시 (`transition: border-color .2s, color .2s`)
- 이미지 페이드: `.35s`
- `all` 전환은 사용하지 않습니다.
